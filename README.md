# Weather Dashboard

This application is used to retrieve the current weather of the selected city.

## User Experience: 

* When the user inputs a city name to search, they are then presented with the city's current weather and the 5 day forecast.

## Resources Used:

* HTML
* CSS
* JavaScript
* Third Party API
